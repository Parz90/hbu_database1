    TRIM (Transact-SQL)

     

    Updated: December 16, 2016

    THIS TOPIC APPLIES TO:SQL Server (starting with vNext)Azure SQL DatabaseAzure SQL Data Warehouse Parallel Data Warehouse

    Removes white spaces or specified characters from a string.

    Syntax

    Copy 
    TRIM ( [ characters FROM ] string )

    Arguments

    characters
    Is a literal, variable, or function call of any non-LOB character type (nvarchar, varchar, nchar, or char) containing characters that should be removed. nvarchar(max) and varchar(max) types are not allowed.

    string
    Is an expression of any character type (nvarchar, varchar, nchar, or char) where characters should be removed.

    Return Types

    Returns a character expression with a type of string argument where white spaces or specified characters are removed from both sides. Returns NULL if input string is NULL.

    Remarks

    By default TRIM function removes white spaces from both sides. This is equivalent to LTRIM(RTRIM(@string)). Behavior of TRIMfunction with specified characters is identical to behavior of REPLACE function where characters from start or end are replaced with empty strings.

    Examples

    A. Removes whitespaces from both sides of string

    The following example removes spaces from before and after the word test.

    Transact-SQL
    Copy 
    SELECT TRIM( '     test    ') AS Result;

    Here is the result set.

    test

    B. Removes specified characters from both sides of string

    The following example removes a trailing period and trailing spaces.

    Transact-SQL
    Copy 
    SELECT TRIM( '.,! ' FROM  '#     test    .') AS Result;

    Here is the result set.
    # test
